to_xls by richardun

This Rails plugin is the easiest way to export to Excel. It gives you the ability to call to_xls to an Array of Hashes. The builder options are the same as to_json / to_xml, except for the :include, :except, and :only. Additionally, there is a :upcase option which allows you to change your columns to upper case. Since you make the Hash, you can add whatever you want in the values or columns, name the columns whatever you want. Why? Because often you have data tables you want to export that are made up of modified and joined data from other tables when you display them, why not export that same modified data set? Also, since the columns read hash names and humanize the values, you can make column names that are more readable, like ":first_name" and will display like: "First name" or "FIRST NAME" (with :upcase option). run
